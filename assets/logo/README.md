# Logos Le Middle

Placez vos fichiers logo ici :

- `logo_black.png` - Logo noir sur fond transparent (400x400px minimum)
- `logo_white.png` - Logo blanc sur fond transparent (400x400px minimum)

Ces fichiers sont utilisés par les générateurs d'images pour :
- Type "Photo" : overlay du logo noir au centre
- Type "Phrase" : logo dans le header de la carte blanche

## Format recommandé
- PNG avec transparence
- Résolution minimale : 400x400 pixels
- Format carré ou proche du carré
